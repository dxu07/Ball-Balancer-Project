\doxysubsection{Detailed Description}
The task that interfaces with the motors. 

This task\textquotesingle{}s main function is to set the duty cycle of either motor while handling external interruptions. This task will be the only task communicating with the \mbox{\hyperlink{motor_8py}{motor.\+py}} and DRV8847.\+py driver modules.
